Instruction: You will be provided a summary of a task performed by a behavior tree, and your objective is to express this behavior tree in XML format.
Input: The behavior tree defines a sequence of actions for a robot. It starts by moving towards a target point, then waits for a specified time before moving to the same or another target point. This sequence is repeated twice.

The "MoveGoal" action takes a target point as input and moves the robot to that location. The "Wait" action takes a wait time as input and pauses the robot's movement for the specified duration.

Overall, this behavior tree orchestrates the robot's movement in a sequential manner, allowing it to navigate to specific points and pause at each point for a defined period before proceeding to the next action.

            </ForceSuccess>
        </Sequence>
