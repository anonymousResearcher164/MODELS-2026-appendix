3. Within DistanceController, execute ROS action ComputePathToPose using the specified goal, outputting the computed path, and setting the planner_id to "GridBased".
4. If ROS action ComputePathToPose returns SUCCESS, proceed to ROS action FollowPath.
5. Execute ROS action FollowPath using the path generated by ComputePathToPose, and set the controller_id to "FollowPath".
6. If any action in the sequence fails, halt the sequence and do not proceed to the next action.
7. If an action in the sequence is still RUNNING, wait for the action to complete before moving to the next action.
